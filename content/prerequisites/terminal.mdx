## 기본 터미널 명령어

터미널은 처음엔 낯설지만, 자주 쓰는 명령어는 10개도 안 돼요.\
아래 명령어만 알아도 바이브코딩을 시작하는 데 충분합니다.

<Callout type="tip">
  명령어를 외울 필요 없어요. 어떤 명령어가 있는지 한 번 훑어보고, 필요할 때 다시 찾아보면 됩니다.
</Callout>

### <FolderOpen size={16} className="inline-block mb-0.5 mr-1.5" /> 폴더 이동 — `cd`

"Change Directory"의 줄임말이에요. 폴더를 이동할 때 씁니다.\
Finder에서 폴더를 더블클릭하는 것과 같아요.

<ShellBlock>
```bash
cd 폴더이름
```
</ShellBlock>

**예시**

<ShellBlock>
```bash
cd Desktop
```
</ShellBlock>

바탕화면 폴더로 이동해요.

<ShellBlock>
```bash
cd my-project
```
</ShellBlock>

`my-project` 폴더 안으로 들어가요.

<ShellBlock>
```bash
cd ..
```
</ShellBlock>

현재 폴더에서 **한 단계 위**로 올라가요. `..`은 "부모 폴더"를 뜻해요.

---

### <List size={16} className="inline-block mb-0.5 mr-1.5" /> 현재 폴더 내용 보기 — `ls`

"List"의 줄임말이에요. 현재 위치에 어떤 파일과 폴더가 있는지 보여줘요.\
Finder에서 폴더를 열어 내용을 보는 것과 같아요.

<ShellBlock>
```bash
ls
```
</ShellBlock>

**출력 예시**

```output
Desktop    Documents    Downloads    my-project    README.md
```

폴더와 파일 이름이 나란히 출력돼요.

---

### <MapPin size={16} className="inline-block mb-0.5 mr-1.5" /> 내가 지금 어디 있는지 확인 — `pwd`

"Print Working Directory"의 줄임말이에요. 현재 내 위치(경로)를 보여줘요.

<ShellBlock>
```bash
pwd
```
</ShellBlock>

**출력 예시**

```output
/Users/john/Desktop/my-project
```

---

### <FolderPlus size={16} className="inline-block mb-0.5 mr-1.5" /> 새 폴더 만들기 — `mkdir`

"Make Directory"의 줄임말이에요. 새 폴더를 만들어요.

<ShellBlock>
```bash
mkdir 폴더이름
```
</ShellBlock>

**예시**

<ShellBlock>
```bash
mkdir my-project
```
</ShellBlock>

`my-project`라는 이름의 폴더가 현재 위치에 만들어져요.

---

### <Eraser size={16} className="inline-block mb-0.5 mr-1.5" /> 터미널 화면 지우기 — `clear`

터미널 화면이 지저분할 때 깨끗하게 비워줘요. 기존 내용이 사라지는 게 아니라 위로 밀려나는 거예요.

<ShellBlock>
```bash
clear
```
</ShellBlock>

## 참고 — 고급 바이브코딩용 터미널

### [cmux](https://github.com/manaflow-ai/cmux/blob/main/README.ko.md)

<Screenshot src="/images/cmux-landing-image.webp" alt="cmux 터미널 앱 화면" />

지금 당장 설치할 필요는 없지만, 나중에 고급 바이브코딩을 할 때 유용한 터미널 앱이에요.

cmux는 클로드코드 같은 AI 코딩 에이전트를 **여러 개 동시에** 운영하는 개발자를 위한 macOS 전용 네이티브 터미널 앱이에요. 세로 탭 사이드바에서 각 세션의 git 브랜치, PR 상태, 최근 알림을 한눈에 볼 수 있고, 에이전트가 입력을 기다릴 때 파란색 알림 링으로 즉시 알려줘 여러 에이전트를 효율적으로 관리할 수 있어요. Swift/AppKit으로 만든 네이티브 앱이라 빠르고 가벼우며, 터미널 옆에 내장 브라우저를 띄워 에이전트가 개발 서버와 직접 상호작용하게 하는 기능도 제공해요.
