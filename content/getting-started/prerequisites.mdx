# 시작 전 준비물

Claude Code를 사용하기 전에 아래 항목들을 준비해두면 훨씬 수월하게 시작할 수 있어요.

## 필수 준비물

### 1. 컴퓨터

- **운영체제:** Windows 10 이상, macOS 10.15 이상, 또는 Linux
- 태블릿이나 스마트폰은 지원하지 않아요

### 2. 인터넷 연결

Claude Code는 Anthropic 서버와 통신하기 때문에 인터넷이 반드시 필요해요.

### 3. Anthropic 계정

Claude Code를 사용하려면 Anthropic 계정으로 로그인해야 해요.

- [claude.ai](https://claude.ai) 에서 무료로 가입할 수 있어요
- 이미 Claude를 사용 중이라면 같은 계정으로 로그인하면 됩니다

<Callout type="info">
  Anthropic 계정은 무료로 만들 수 있지만, Claude Code 사용량에 따라 요금이 부과될 수 있어요. 요금 정책은 [anthropic.com](https://anthropic.com)에서 확인하세요.
</Callout>

### 4. 터미널 (명령어 창)

Claude Code는 터미널에서 실행해요. 터미널은 마우스 대신 키보드로 명령을 내리는 창이에요.

- **Mac:** 기본 설치되어 있어요 (`Cmd + Space` → "터미널" 검색)
- **Windows:** 명령 프롬프트 또는 PowerShell (`Win + R` → `cmd`)
- **Linux:** 기본 터미널 앱 사용

<Callout type="tip">
  터미널이 처음이라 낯설어도 괜찮아요! Claude Code와 대화할 때 필요한 명령어는 몇 가지뿐이에요.
</Callout>

## 선택 준비물

### 작업할 프로젝트 폴더

만들고 싶은 것이 있다면, 작업 파일을 보관할 폴더를 미리 만들어두세요. 바탕화면에 새 폴더를 하나 만들어도 충분합니다.

### Git (버전 관리 도구)

Git을 사용하면 실수로 파일이 수정되더라도 이전 상태로 되돌릴 수 있어요. 필수는 아니지만 있으면 안전해요.

<Callout type="warning">
  Git이 없어도 Claude Code를 사용할 수 있지만, 중요한 파일을 수정하기 전에는 반드시 다른 곳에 백업해두세요.
</Callout>
