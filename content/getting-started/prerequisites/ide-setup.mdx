# IDE 설치하기 (Antigravity)

CLI 명령어가 불편하신 분들을 위해 GUI 형식으로도 바이브코딩을 할 수 있습니다.\
**IDE(Integrated Development Environment, 통합 개발 환경)** 는 코드 편집기, 디버거, 터미널, AI 어시스턴트 등 소프트웨어 개발에 필요한 도구들을 하나의 프로그램 안에 통합해 놓은 GUI 애플리케이션입니다.\
한 마디로, "코딩에 필요한 모든 것을 한 지붕 아래 모아 놓은 작업 공간"입니다.\
이 가이드에서는 **[Google Antigravity(앤티그래비티)](https://antigravity.google/)** 를 사용합니다. [Antigravity](https://antigravity.google/)는 업계 표준 수준의 확장 프로그램 생태계와 가벼운 성능을 자랑하는 VSCode 기반의 에이전트 중심 IDE입니다.\
VSCode의 모든 기능을 지원하면서 Gemini 3 기반의 Google 생태계 작업에 특히 강점이 있습니다.

<Callout type="info">
  클로드코드는 Antigravity의 Extensions 사이드바를 통해 IDE 안에서 바로 실행할 수 있습니다.
</Callout>

 

## [Antigravity](https://antigravity.google/) 설치 및 실행

[**antigravity.google**](http://antigravity.google)에 접속한 후 macOS용 설치 파일을 다운로드하세요.\
다운로드 용량은 약 200~300MB 정도입니다.\
macOS에서는 열린 창에서 Antigravity 아이콘을 **Applications 폴더로 드래그**하면 설치가 완료됩니다.\
설치 파일을 실행하세요.
