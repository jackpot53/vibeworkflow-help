# 클로드 구독하기

Claude Code는 Claude 유료 구독이 있어야 사용할 수 있습니다.\
[https://claude.ai/upgrade](https://claude.ai/upgrade) 에서 결제하고 구독합니다.

## 플랜 비교

| 플랜 | 가격 | Claude Code | 특징 |
|------|------|-------------|------|
| Free | 무료 | ❌ 불가 | 채팅만 가능 (코딩 에이전트 없음) |
| Pro | $20/월 | ✅ 가능 (기본) | 개인 개발자용, 사용량 제한 있음 |
| Max 5x | $100/월 | ✅ 가능 (확장) | Pro 대비 약 5배 사용량 |
| Max 20x | $200/월 | ✅ 가능 (풀 사용) | 매우 높은 사용량, 헤비 유저용 |
| Team | $30~125/인/월 | ✅ 가능 | 협업 + 관리자 기능 |

> **추천:** 처음엔 **Pro($20/월)** 로 시작하고, 사용량이 부족하다고 느껴지면 **Max 5x($100/월)** 로 업그레이드하세요.\
> FDE는 $100/월(Max 5x)을 구독해서 사용하고 있습니다.

## 사용량 제한이란?

Claude Code는 AI 모델을 호출하는 만큼 비용이 발생하는 구조입니다.\
플랜마다 일정 주기(보통 5시간) 안에 쓸 수 있는 양이 정해져 있습니다.

- 제한에 도달하면 Claude Code가 잠시 응답을 멈추고, 일정 시간 후 자동으로 초기화됩니다.
- 한도를 자주 초과한다면 상위 플랜 업그레이드를 고려하세요.

## 구독 방법

1. [https://claude.ai/upgrade](https://claude.ai/upgrade) 접속
2. 원하는 플랜 선택 (처음이라면 **Pro** 권장)
3. 신용카드 또는 체크카드로 결제
4. 구독 완료 후 [claude.ai](https://claude.ai) 에서 로그인 상태 확인

## 구독 확인

구독이 완료되면 [claude.ai](https://claude.ai) 우측 상단 프로필에서 현재 플랜을 확인할 수 있습니다.

> **결제 참고:** 매월 구독일 기준으로 자동 갱신됩니다. 언제든지 설정에서 해지할 수 있습니다.
