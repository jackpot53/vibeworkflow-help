# 터미널 설치하기

## [iTerm2](https://iterm2.com/) 설치

<Screenshot src="https://velog.velcdn.com/images/xangj0ng/post/093f057a-3b55-4f47-9951-70f0715361ea/image.png" alt="iTerm2 터미널 화면" />

클로드코드로 바이브코딩을 하려면 터미널에서 명령어 + 자연어를 많이 입력해야 해요.

| 항목 | macOS 기본 터미널 | [iTerm2](https://iterm2.com/) |
|------|------------------|-------------------------------|
| 속도 | 보통 | 빠름 |
| 탭 · 분할 창 | 제한적 | 자유롭게 여러 창 분할 가능 |
| 자동완성 | 없음 | 지원 |
| 검색 | 기본 수준 | 강력한 텍스트 검색 |
| 커스터마이징 | 제한적 | 색상·폰트·단축키 등 세밀하게 설정 가능 |
| 바이브코딩 적합성 | 🔺 쓸 수는 있음 | ✅ 추천 |

다음 사이트에서 다운로드 하고 설치해요

## 참고 — 고급 바이브코딩용 터미널

### [cmux](https://github.com/manaflow-ai/cmux/blob/main/README.ko.md)

<Screenshot src="https://cmux.com/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Flanding-image.70ebd164.png&w=3840&q=75&dpl=dpl_71HzjtY7niv3kQHG4zp1yeNgdEUx" alt="cmux 터미널 앱 화면" />

지금 당장 설치할 필요는 없지만, 나중에 고급 바이브코딩을 할 때 유용한 터미널 앱이에요.

cmux는 Claude Code 같은 AI 코딩 에이전트를 **여러 개 동시에** 운영하는 개발자를 위한 macOS 전용 네이티브 터미널 앱이에요. 세로 탭 사이드바에서 각 세션의 git 브랜치, PR 상태, 최근 알림을 한눈에 볼 수 있고, 에이전트가 입력을 기다릴 때 파란색 알림 링으로 즉시 알려줘 여러 에이전트를 효율적으로 관리할 수 있어요. Swift/AppKit으로 만든 네이티브 앱이라 빠르고 가벼우며, 터미널 옆에 내장 브라우저를 띄워 에이전트가 개발 서버와 직접 상호작용하게 하는 기능도 제공해요.
