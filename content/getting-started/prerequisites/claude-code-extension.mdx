# 클로드 코드 익스텐션 설치하기

Antigravity는 전 세계에서 가장 많이 쓰이는 코드 편집기인 **VS Code를 기반으로 만들어진 편집기**예요.

<Callout type="info">
  **포크(fork)가 뭔가요?** 기존 프로그램의 소스코드를 그대로 가져와서, 거기에 새로운 기능을 얹어 새 프로그램을 만드는 방식이에요. 레시피를 그대로 가져다가 재료를 몇 가지 더 추가한 나만의 요리를 만드는 것과 비슷해요. Antigravity는 VS Code라는 레시피를 기반으로, AI 기능을 강화해서 만든 편집기예요.
</Callout>

덕분에 VS Code용으로 만들어진 수천 개의 확장 프로그램을 Antigravity에서도 그대로 쓸 수 있어요.\
클로드코드 익스텐션도 그 중 하나로, 설치하면 편집기를 벗어나지 않고 Claude와 대화하면서 코딩할 수 있어요.\
하나의 화면에서 Gemini와 Claude, 두 AI를 동시에 활용할 수도 있습니다.

## 확장 프로그램(익스텐션)이란?

확장 프로그램은 편집기에 **추가로 설치하는 기능 모음**이에요.\
스마트폰에서 기본 앱 외에 카카오톡, 유튜브 같은 앱을 추가로 설치하는 것과 똑같은 개념이에요.

Antigravity 자체는 코드를 작성하고 보여주는 기본 편집기예요.\
여기에 클로드코드 확장 프로그램을 설치하면, 편집기 안에서 바로 Claude와 대화하며 코딩할 수 있는 기능이 추가되는 거예요.

## 클로드코드 검색 & 설치

왼쪽 사이드바의 **익스텐션 패널 열기 아이콘**(<img src="https://raw.githubusercontent.com/microsoft/vscode-icons/main/icons/dark/extensions.svg" style={{display:'inline', height:'1em', verticalAlign:'middle', filter:'invert(1)'}} /> 네 개의 블록이 퍼즐처럼 맞물린 모양)을 클릭하세요.

<Callout type="info">
  단축키 `Cmd+Shift+X`(Mac) / `Ctrl+Shift+X`(Windows)로도 바로 열 수 있습니다.
</Callout>

패널 상단 검색창에 `claude code`를 입력하세요.

![클로드코드 검색 화면](/images/vscode-extension-install.jpg)

검색 결과 목록에서 **Claude Code for VS Code** (게시자: Anthropic)를 클릭한 뒤, **Install** 버튼을 클릭해 설치를 완료하세요.

설치가 끝나면 다른 프로그램을 열지 않아도 Antigravity 안에서 바로 Claude와 대화하며 코딩을 시작할 수 있어요.


## 클로드코드 익스텐션 패널 열기

설치가 완료되면 Spark 아이콘(<img src="https://mintcdn.com/claude-code/c5r9_6tjPMzFdDDT/images/vs-code-spark-icon.svg?fit=max&auto=format&n=c5r9_6tjPMzFdDDT&q=85&s=3ca45e00deadec8c8f4b4f807da94505" style={{display:'inline', height:'1em', verticalAlign:'middle'}} />)이 여러 곳에 나타납니다.
클로드코드를 여는 방법은 다음과 같습니다.

**편집기 도구 모음 (가장 빠른 방법)** : 파일을 열었을 때 편집기 오른쪽 위 모서리에 나타나는 Spark 아이콘을 클릭하세요.

![편집기 도구 모음의 Spark 아이콘](https://mintcdn.com/claude-code/mfM-EyoZGnQv8JTc/images/vs-code-editor-icon.png?fit=max&auto=format&n=mfM-EyoZGnQv8JTc&q=85&s=eb4540325d94664c51776dbbfec4cf02)

**그 외 접근 방법**

| 방법 | 위치 | 설명 |
|------|------|------|
| **활동 표시줄** | 왼쪽 사이드바 | Spark 아이콘(<img src="https://mintcdn.com/claude-code/c5r9_6tjPMzFdDDT/images/vs-code-spark-icon.svg?fit=max&auto=format&n=c5r9_6tjPMzFdDDT&q=85&s=3ca45e00deadec8c8f4b4f807da94505" style={{display:'inline', height:'1em', verticalAlign:'middle'}} />) 클릭 → 세션 목록에서 기존 세션을 열거나 새 세션 시작 |
| **명령 팔레트** | `Cmd+Shift+P` / `Ctrl+Shift+P` | "클로드코드" 검색 → "새 탭에서 열기" 선택 |
| **상태 표시줄** | 창 오른쪽 아래 모서리 | Spark 아이콘(<img src="https://mintcdn.com/claude-code/c5r9_6tjPMzFdDDT/images/vs-code-spark-icon.svg?fit=max&auto=format&n=c5r9_6tjPMzFdDDT&q=85&s=3ca45e00deadec8c8f4b4f807da94505" style={{display:'inline', height:'1em', verticalAlign:'middle'}} />) 클릭 — 파일을 열지 않은 상태에서도 작동 |

![클로드코드 익스텐션 전체 패널](https://github.com/jackpot53/books/blob/main/claude-code-extension.png?raw=true)

<Callout type="tip">
  Claude 패널은 드래그로 원하는 위치에 자유롭게 재배치할 수 있습니다.
</Callout>

## 이제 GUI로 바이브코딩!

클로드코드는 원래 터미널에서 명령어를 입력하는 **CLI(Command Line Interface) 도구**예요.\
하지만 익스텐션을 설치하면 터미널 없이도 **편집기 안의 패널에서 마우스와 키보드만으로** Claude와 대화하며 코딩할 수 있어요.

<Callout type="tip">
  CLI를 몰라도 괜찮아요. 익스텐션 패널이 터미널의 역할을 대신하기 때문에, 명령어 없이도 바이브코딩의 모든 기능을 그대로 사용할 수 있습니다.
</Callout>

![Antigravity에서 클로드코드 GUI 패널로 바이브코딩하는 화면](/images/claude-code-extension.png)

## 온보딩 체크리스트 확인

패널을 처음 열면 **Learn 클로드코드** 체크리스트가 나타납니다.\
**Show me**를 클릭해 각 항목을 진행하거나, 필요 없으면 X로 닫아도 됩니다.

<Callout type="info">
  나중에 다시 열려면 Extensions → 클로드코드 설정에서 **Hide Onboarding**을 선택 해제하세요.
</Callout>
