# [brew(Homebrew)](https://brew.sh/ko/) 설치

[Homebrew](https://brew.sh/ko/)(홈브루, 줄여서 "브루")는 macOS와 Linux에서 사용하는 패키지 관리자입니다.<br/>
쉽게 말하면, 프로그램을 설치·업데이트·삭제를 명령어 한 줄로 관리해주는 도구예요.

즉, 앱스토어 없이 터미널로 모든 개발 도구를 관리할 수 있습니다.<br/>
개발하다가 필요한 프로그램이나 패키지들을 brew를 통해 설치할 경우가 있으니 설치를 해야 합니다.

 MacOS의 `Terminal` 앱을 열고 다음 명령어를 붙여넣기 한다음 `enter` 를 해서 설치합니다.

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## 설치 확인

설치가 완료되면 다음 명령어로 정상 설치 여부를 확인합니다.

```shell
brew --version
```

아래와 같이 버전 정보가 출력되면 설치 성공입니다.

```output
Homebrew 4.x.x
```

혹시 `command not found: brew` 오류가 뜬다면, 터미널을 완전히 종료 후 다시 열어보세요.
