## 히어로 섹션 꾸미기

**히어로 섹션(Hero Section)** 은 웹사이트에 처음 접속했을 때 가장 먼저 보이는 최상단 영역이에요. 보통 서비스 이름, 한 줄 소개 문구, CTA(Call to Action) 버튼으로 구성되고, 방문자의 첫인상을 결정하는 가장 중요한 공간이에요.

이 히어로 섹션에 움직이는 애니메이션을 넣으면 첫인상이 훨씬 풍부해져요. [LottieFiles](https://lottiefiles.com/)는 무료로 사용할 수 있는 고품질 애니메이션을 제공하는 서비스예요.

## Grid로 UI 배치하기

**Grid**는 화면을 격자(열·행)로 나눠서 여러 요소를 일정하게 배치하는 레이아웃 방식이에요. Features 카드, 팀 소개, 포트폴리오 갤러리처럼 같은 형태의 항목이 반복될 때 특히 유용해요.

| 패턴 | 설명 | 주로 쓰이는 섹션 |
|------|------|----------------|
| 2열 Grid | 화면을 절반씩 나눔. 이미지+텍스트 대비 구도에 적합 | About, 기능 소개 |
| 3열 Grid | 균형 잡힌 카드 나열. 가장 많이 쓰이는 패턴 | Features, 서비스 소개 |
| 4열 Grid | 작은 카드를 촘촘하게 배치 | 팀 소개, 로고 모음, 갤러리 |
| auto-fill Grid | 화면 너비에 따라 열 수가 자동으로 늘고 줄어듦 | 반응형 카드 목록 |

## 웹폰트 적용하기

기본 시스템 폰트 대신 [Google Fonts](https://fonts.google.com/)를 적용하면 사이트 분위기가 확 달라져요.

### 1단계: 폰트 고르고 Embed 코드 복사하기

[fonts.google.com](https://fonts.google.com/)에서 원하는 폰트를 선택한 뒤 **Get embed code** 버튼을 클릭하세요. **Embed code in the `<head>` of your html** 항목의 `<link>` 태그 전체를 복사하면 돼요.

### 2단계: Claude Code에 전달하기

복사한 embed 코드를 아래 프롬프트 뒤에 그대로 붙여넣어서 전달하세요.

**사이트 전체에 적용할 때**

<VibeBlock>
```
이 폰트를 사이트 전체에 적용해줘.
[복사한 embed 코드를 여기에 붙여넣기]
```
</VibeBlock>

**특정 영역에만 적용할 때**

<VibeBlock>
```
이 폰트를 제목에만 적용해줘.
[복사한 embed 코드를 여기에 붙여넣기]
```
</VibeBlock>

## 스킬 카드 만들기

<VibeBlock>
```
스킬 섹션에 카드들을 2열 그리드로 배치해줘.
각 카드는 둥근 모서리에 테두리 있고, 호버하면 그림자가 진해져.
카드 안에는 컬러 아이콘 + 카테고리 이름, 설명 텍스트,
아래에 #태그 형태로 기술 목록 나열해줘.
```
</VibeBlock>

## 타임라인 섹션 만들기

경력, 프로젝트 히스토리, 작업 순서를 시각적으로 나열할 때 사용해요. 세로선과 점으로 흐름을 만들고, 주요 항목은 색상으로 강조할 수 있어요.

<VibeBlock>
```
프로젝트 섹션을 타임라인 형식으로 만들어줘.
왼쪽에 세로선 긋고, 각 항목마다 동그란 점 표시해줘.
대표 프로젝트는 초록색으로 강조하고 별 아이콘 넣어줘.
나머지는 번호로.
각 카드 안에는 프로젝트 이름, GitHub 링크, 설명, 해시태그 목록 넣어줘.
```
</VibeBlock>
