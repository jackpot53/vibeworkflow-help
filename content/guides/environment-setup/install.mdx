# 클로드코드 CLI 설치하기

<Screenshot src="https://vld-bc.com/uploads/Claude_Code_Title_a399fa2c8b.png" alt="클로드코드 CLI" />

**CLI(Command Line Interface)** 란 마우스 클릭 대신 텍스트 명령어를 입력해서 프로그램을 실행하는 방식이에요.\
GUI(그래픽 인터페이스)가 Finder에서 폴더를 클릭하는 방식이라면, CLI는 터미널에 `cd my-project`처럼 직접 입력하는 방식이에요.

클로드코드 CLI는 이 터미널에서 `claude` 명령어 하나로 실행하는 AI 개발 도구예요.\
코드 읽기, 수정, 실행까지 자연어로 AI와 함께 할 수 있습니다.

<Steps>

### Node.js 설치

클로드코드 CLI를 설치하려면 **Node.js**와 **npm**이 필요합니다.

> **Node.js** — 자바스크립트를 실행하는 엔진 (실행 환경)\
> **npm** (Node Package Manager) — 필요한 라이브러리를 설치하고 관리하는 도구

<Callout type="info">
  Node.js를 설치하면 npm은 자동으로 함께 설치됩니다.
</Callout>

다음 링크에서 [Node.js](https://nodejs.org/ko/download)를 다운로드하여 설치합니다.

설치 후 터미널에서 아래 명령어로 정상 설치 여부를 확인합니다.

<ShellBlock>
```bash
node --version
```
</ShellBlock>

<ShellBlock>
```bash
npm --version
```
</ShellBlock>

버전 숫자가 출력되면 설치 완료입니다.

### 클로드코드 CLI 설치 및 실행

클로드코드 CLI는 로컬에 설치되어 터미널 기반으로 내 코드를 이해하고, 직접 수정·실행까지 해주는 AI 개발 도구입니다.

<Tabs items={['Native Install (권장)', 'npm으로 설치', 'Homebrew으로 설치']}>

<Tabs.Tab>

터미널에 다음 명령어를 입력합니다.

<ShellBlock>
```bash
curl -fsSL https://claude.ai/install.sh | bash
```
</ShellBlock>

설치가 완료되면 아래와 같은 메시지가 출력됩니다.

```output
✔ Claude Code successfully installed!
  Version: 2.1.50
  Location: ~/.local/bin/claude
  Next: Run claude --help to get started

⚠ Setup notes:
  • Native installation exists but ~/.local/bin is not in your PATH. Run:

  echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

✅ Installation complete!
```

**PATH 설정** : `claude` 명령어를 어느 위치의 터미널에서든 실행할 수 있도록 PATH를 등록합니다.

<ShellBlock>
```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
```
</ShellBlock>

**설치 확인** : 다음 명령어 입력 시 버전이 표시되면 설치가 완료된 것입니다.

<ShellBlock>
```bash
claude -v
```
</ShellBlock>

```output
2.1.87 (Claude Code)
```

</Tabs.Tab>

<Tabs.Tab>

터미널에 다음 명령어를 입력합니다.

<ShellBlock>
```bash

```
</ShellBlock>

**설치 확인** : 다음 명령어 입력 시 버전이 표시되면 설치가 완료된 것입니다.

<ShellBlock>
```bash
claude -v
```
</ShellBlock>

```output
2.1.87 (Claude Code)
```

</Tabs.Tab>

<Tabs.Tab>

터미널에 다음 명령어를 입력합니다.

<ShellBlock>

</ShellBlock>

**설치 확인** : 다음 명령어 입력 시 버전이 표시되면 설치가 완료된 것입니다.

<ShellBlock>
```bash
claude -v
```

### 첫 실행 및 초기 설정

작업할 프로젝트 폴더로 이동한 뒤 `claude` 명령어를 입력합니다.

<ShellBlock>
```bash
cd your-project   # 작업할 프로젝트 폴더로 이동
claude
```
</ShellBlock>

<Callout type="info">
  `your-project` 자리에 본인이 작업할 폴더 경로를 입력하세요.
</Callout>

### 터미널 상태 표시줄 설정

클로드코드는 터미널 상단에 상태 표시줄을 띄워 현재 세션 정보를 실시간으로 확인할 수 있습니다.

상태 표시줄이 유용한 경우:
- 컨텍스트 윈도우 사용량을 모니터링할 때
- 세션 비용을 추적할 때
- Git 브랜치 및 상태를 항상 확인하고 싶을 때

<Screenshot src="/images/statusline-annotated.jpg" alt="터미널 상태 표시줄 예시 — 빨간 테두리 영역이 상태 표시줄" />

클로드코드 CLI 안에서 [`/statusline`](/reference/commands/statusline) 명령어에 원하는 내용을 자연어로 입력합니다.

<SlashBlock>
```
/statusline 모델 이름, 컨텍스트 진행 상태 바, 현재 디렉토리, Git 브랜치, 그리고 현재 시간을 표시해라
```
</SlashBlock>

설정이 완료되면 다음과 같이 확인 메시지가 출력됩니다.

```output
⏺ Your statusline is configured! It will display:

  - Model name (cyan)
  - Context progress bar with percentage
  - Current directory (yellow, ~ for home)
  - Git branch in parentheses (yellow)
  - Current time HH:MM:SS (green)

  The configuration is saved in ~/.claude/settings.json
  with the script at ~/.claude/statusline-command.sh.
```

<Callout type="info">
  설정 후 `/exit`으로 세션을 나갔다가 `claude`로 다시 입력하면 상태 표시줄이 활성화됩니다.
</Callout>

<Callout type="tip">
  스크립트 샘플이나 표시 항목 커스터마이징 방법은 [/statusline 가이드](/reference/commands/statusline)를 참고하세요.
</Callout>

</Steps>
