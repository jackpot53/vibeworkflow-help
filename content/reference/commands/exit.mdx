---

## 사용법

```
> /exit
```

실행하면 세션이 바로 종료되고 일반 터미널 화면으로 돌아와요.

<Callout type="info">
  `/exit` 대신 **Ctrl + C** 또는 **Ctrl + D**를 눌러도 동일하게 종료돼요.\
  `/quit`을 입력해도 같은 동작을 해요.
</Callout>

---

## 파일은 자동으로 저장되나요?

네. Claude Code가 수정한 파일은 **자동으로 저장**돼요.\
종료 전에 따로 저장 명령을 실행할 필요 없어요.

단, 아래 항목은 종료 전에 직접 확인하세요:

| 확인 항목 | 이유 |
|-----------|------|
| Git 커밋 | Claude가 수정한 내용은 저장되지만 커밋은 자동으로 되지 않아요 |
| 진행 중인 작업 | 긴 작업을 실행 중이라면 완료 후 종료하세요 |

---

## 관련 명령어

| 명령어 | 설명 |
|--------|------|
| `/clear` | 종료하지 않고 대화만 초기화 |
| `/status` | 종료 전 현재 세션 상태 확인 |
