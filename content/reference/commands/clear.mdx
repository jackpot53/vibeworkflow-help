# /clear

---

## 언제 쓰나요?

새로운 작업을 시작하거나, Claude가 이전 대화 내용 때문에 엉뚱한 답변을 반복할 때 사용해요.

**이런 상황에서 쓰세요:**

- 지금까지 하던 작업이 완전히 끝나고 전혀 다른 작업을 시작할 때
- Claude가 자꾸 잘못된 방향으로 대답할 때 (이전 맥락이 방해하는 경우)
- 대화가 너무 길어져서 응답이 느리거나 이상해졌을 때

---

## 사용법

```
> /clear
```

실행하면 대화 기록이 즉시 사라지고 빈 화면으로 돌아와요.

<Callout type="warning">
  `/clear` 후에는 이전 대화 내용을 복구할 수 없어요.\
  작업을 이어가야 한다면 `/compact`를 먼저 써보세요.
</Callout>

---

## `/compact`과 뭐가 달라요?

| | `/clear` | `/compact` |
|--|----------|------------|
| 대화 기록 | 완전히 삭제 | 요약해서 보존 |
| 지금까지의 맥락 | 사라짐 | 요약본으로 유지됨 |
| 언제 쓰나 | 새 작업 시작할 때 | 작업을 이어갈 때 |

---

## 관련 명령어

| 명령어 | 설명 |
|--------|------|
| `/compact` | 대화를 요약해서 토큰 절약 (맥락 유지) |
| `/context` | 현재 컨텍스트 사용량 확인 |
| `/rewind` | 특정 시점으로 되돌리기 |
