# /statusline

<CommandCard
  command="/statusline"

---

## 언제 쓰나요?

터미널에서 Claude Code를 사용할 때 현재 모델, 토큰 사용량, 작업 상태 등을 한눈에 확인하고 싶을 때 사용해요.

---

## 사용법

```
> /statusline
```

인자 없이 실행하면 현재 설정을 보여줘요.

```
> /statusline auto
> /statusline off
```

---

## 옵션

| 옵션 | 설명 |
|------|------|
| `auto` | 터미널이 지원하면 자동으로 상태표시줄을 표시해요 (기본값) |
| `off` | 상태표시줄을 끄고 싶을 때 사용해요 |

<Callout type="info">
  상태표시줄에는 현재 사용 중인 **모델명**, **토큰 사용량**, **작업 진행 상태** 등이 표시돼요.\
  터미널 종류에 따라 지원 여부가 다를 수 있어요.
</Callout>
