## 언제 쓰나요?

터미널에서 Claude Code를 사용할 때 현재 모델, 컨텍스트 사용량, 비용 등을 한눈에 확인하고 싶을 때 사용해요.

---

## 사용법

자연어로 원하는 내용을 설명하면 Claude가 스크립트를 자동으로 만들어줘요.

```
> /statusline show model name and context percentage
> /statusline 모델명이랑 컨텍스트 사용량 보여줘
> /statusline show current directory, git branch, model, and context usage
> /statusline git 브랜치랑 모델명, 컨텍스트 사용량 보여줘
```

---

## 설정 방식

상태표시줄은 `settings.json`에 쉘 스크립트를 연결하는 방식으로 동작해요.\
Claude Code가 실행될 때마다 해당 스크립트를 호출하고, 출력 결과를 상태표시줄에 표시해요.

```json
// ~/.claude/settings.json
{
  "statusLine": {
    "type": "command",
    "command": "bash ~/.claude/statusline.sh",
    "refreshInterval": 5
  }
}
```

| 옵션 | 설명 |
|------|------|
| `type` | `"command"` 고정 |
| `command` | 실행할 쉘 스크립트 경로 |
| `refreshInterval` | 갱신 주기 (초, 1–60) |

---

## 스크립트에서 사용할 수 있는 데이터

스크립트는 **stdin으로 JSON을 받아요**. 주요 필드는 다음과 같아요:

| 필드 | 설명 |
|------|------|
| `model.display_name` | 현재 사용 중인 모델명 |
| `context_window.used_percentage` | 컨텍스트 사용률 (0–100) |
| `context_window.remaining_percentage` | 컨텍스트 잔여율 |
| `total_input_tokens` | 누적 입력 토큰 수 |
| `total_output_tokens` | 누적 출력 토큰 수 |
| `cost.total_cost_usd` | 누적 비용 (USD) |
| `workspace.current_dir` | 현재 작업 디렉터리 |

---

## 설정 샘플

### 모델명 + 컨텍스트 사용량

```bash
#!/usr/bin/env bash
input=$(cat)

model=$(echo "$input" | jq -r '.model.display_name // "?"')
used=$(echo "$input"  | jq -r '.context_window.used_percentage // 0' | cut -d. -f1)

echo "[${model}] 컨텍스트: ${used}%"
```

출력 예시:
```
[claude-sonnet-4-6] 컨텍스트: 23%
```

---

### 모델명 + 컨텍스트 + 비용

```bash
#!/usr/bin/env bash
input=$(cat)

model=$(echo "$input" | jq -r '.model.display_name // "?"')
used=$(echo "$input"  | jq -r '.context_window.used_percentage // 0' | cut -d. -f1)
cost=$(echo "$input"  | jq -r '.cost.total_cost_usd // "0.0000"')

echo "[${model}]  컨텍스트: ${used}%  비용: \$${cost}"
```

출력 예시:
```
[claude-sonnet-4-6]  컨텍스트: 42%  비용: $0.0032
```

### 컬러 프로그레스바 (고급)

컨텍스트 사용량에 따라 색상이 바뀌는 바를 표시해요.

```bash
#!/usr/bin/env bash
input=$(cat)

RED="\033[31m"; YELLOW="\033[33m"; GREEN="\033[32m"; RESET="\033[0m"

model=$(echo "$input" | jq -r '.model.display_name // "?"')
used=$(echo "$input"  | jq -r '.context_window.used_percentage // 0' | cut -d. -f1)

if   [ "$used" -ge 80 ]; then color=$RED
elif [ "$used" -ge 50 ]; then color=$YELLOW
else                          color=$GREEN
fi

printf "[%s] 컨텍스트: ${color}%s%%${RESET}\n" "$model" "$used"
```

출력 예시 (80% 이상이면 빨간색):

```
[claude-opus-4-7] 컨텍스트: 87%   ← 빨간색으로 표시
```

### 디렉터리 + Git 브랜치 + 모델명 + 컨텍스트 (종합)

현재 폴더, Git 브랜치, 모델, 컨텍스트 사용량을 한 줄에 모두 표시해요.

```bash
#!/usr/bin/env bash
input=$(cat)

BLUE="\033[34m"; GREEN="\033[32m"; MAGENTA="\033[35m"
YELLOW="\033[33m"; RED="\033[31m"; RESET="\033[0m"; BOLD="\033[1m"

cwd=$(echo "$input"   | jq -r '.workspace.current_dir // empty')
model=$(echo "$input" | jq -r '.model.display_name // "?"')
used=$(echo "$input"  | jq -r '.context_window.used_percentage // 0' | cut -d. -f1)

# 현재 디렉터리에서 git 브랜치 가져오기
branch=""
if [ -n "$cwd" ]; then
  branch=$(git -C "$cwd" branch --show-current 2>/dev/null)
fi

# 컨텍스트 사용량에 따라 색상 결정
if   [ "$used" -ge 80 ]; then ctx_color=$RED
elif [ "$used" -ge 50 ]; then ctx_color=$YELLOW
else                          ctx_color=$GREEN
fi

dir_part="${BLUE}${BOLD}$(basename "$cwd")${RESET}"
branch_part="${GREEN}${branch}${RESET}"
model_part="${MAGENTA}${model}${RESET}"
ctx_part="${ctx_color}${used}%${RESET}"

printf "%s  %s  [%s]  ctx: %s\n" "$dir_part" "$branch_part" "$model_part" "$ctx_part"
```

출력 예시:
```
my-project  main  [claude-sonnet-4-6]  ctx: 23%
```

브랜치가 `main`이 아닐 때는 초록색으로 강조돼서 한눈에 확인할 수 있어요.

<Callout type="tip">
  스크립트에서 `jq`를 사용하므로 `jq`가 설치되어 있어야 해요.\
  `brew install jq`로 설치할 수 있어요.
</Callout>
