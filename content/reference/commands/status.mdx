## 언제 쓰나요?

지금 어떤 모델을 쓰고 있는지, Claude Code가 제대로 연결돼 있는지 빠르게 확인하고 싶을 때 사용해요.

---

## 사용법

```
> /status
```

실행하면 현재 세션 정보가 출력돼요:

```
  Version:          2.1.138
  Session name:     Update sidebar menu labels for vibe-coding-intro plan page
  Session ID:       44c6cefc-7dd9-4fee-a346-ca5a5619d9c7
  cwd:              /Users/amiz/dev/vibeworkflow-help
  Login method:     Claude Max account
  Organization:     john.k@kakaocorp.com's Organization
  Email:            john.k@kakaocorp.com

  Model:            opusplan (claude-sonnet-4-6)
  MCP servers:      1 need auth, 1 disabled · /mcp
  Setting sources:  User settings, Shared project settings, Project local settings
  Esc to cancel
```

---

## 표시 항목

| 항목 | 설명 |
|------|------|
| **Version** | 현재 설치된 Claude Code 버전 |
| **Session name** | 현재 세션의 이름 (대화 내용 기반으로 자동 생성) |
| **Session ID** | 세션 고유 식별자 |
| **cwd** | Claude Code가 실행 중인 현재 디렉토리 |
| **Login method** | 로그인 방식 (Claude Max account / API key 등) |
| **Organization** | 소속 조직 |
| **Email** | 로그인된 계정 이메일 |
| **Model** | 현재 사용 중인 모델 (플랜명과 실제 모델 ID) |
| **MCP servers** | 연결된 MCP 서버 상태 |
| **Setting sources** | 적용된 설정 파일 출처 |

<Callout type="info">

</Callout>
