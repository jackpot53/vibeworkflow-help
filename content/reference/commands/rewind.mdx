## 언제 쓰나요?

Claude가 여러 단계에 걸쳐 작업하다가 방향이 잘못됐을 때, 특정 시점으로 되돌리고 싶을 때 사용해요.

**이런 상황에서 쓰세요:**

- 몇 가지 수정을 했는데 결과가 마음에 들지 않아서 여러 단계 전으로 돌아가고 싶을 때
- "저번에 그렇게 했을 때가 더 나았는데" 싶은 시점으로 되돌리고 싶을 때

---

## 사용법

`/rewind`를 입력하거나 **Esc를 두 번** 눌러 메뉴를 열어요.

**1단계 — 시점 선택**

세션 중 입력했던 프롬프트 목록이 표시돼요.\
**↑↓ 화살표 키**로 되돌릴 시점을 선택하고 **Enter**를 눌러요.

**2단계 — 옵션 선택**

이어서 아래 4가지 옵션이 표시돼요.\
마찬가지로 **↑↓ 화살표 키**로 선택하고 **Enter**를 눌러요.

| 옵션 | 설명 |
|------|------|
| **Restore code and conversation** | 파일과 대화 내용을 모두 되돌려요 |
| **Restore conversation** | 대화만 되돌리고 파일은 현재 상태 유지 |
| **Restore code** | 파일만 되돌리고 대화는 현재 상태 유지 |
| **Summarize from here** | 해당 시점 이후 대화를 요약해서 압축 |
| **Never mind** | 아무것도 되돌리지 않고 취소 |

<Callout type="warning">
  체크포인트는 Claude의 파일 편집 도구로 수정된 파일만 추적해요.\
  터미널에서 직접 실행한 Bash 명령어로 바뀐 파일은 되돌릴 수 없어요.
</Callout>

---

## 체크포인트는 어떻게 만들어지나요?

Claude가 응답할 때마다 **자동으로 스냅샷**이 저장돼요. 따로 설정할 필요 없어요.\
변경된 파일만 증분 저장하므로 공간도 효율적으로 사용해요.
