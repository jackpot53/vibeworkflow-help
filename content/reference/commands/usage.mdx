## 언제 쓰나요?

이번 세션에서 비용이 얼마나 나왔는지, 플랜 한도가 얼마나 남았는지 확인하고 싶을 때 사용해요.

## 표시 항목

**Session — 이번 세션 통계**

| 항목 | 설명 |
|------|------|
| **Total cost** | 이번 세션에서 사용한 총 비용 |
| **Total duration (API)** | 실제 API 응답에 걸린 시간 |
| **Total duration (wall)** | 세션을 열어둔 실제 경과 시간 |
| **Total code changes** | 추가·삭제된 코드 줄 수 |
| **Usage by model** | 모델별 input/output/cache 토큰 수와 비용 |

**한도 사용률**

| 항목 | 설명 |
|------|------|
| **Current session** | 현재 세션의 한도 사용률과 초기화 시간 |
| **Current week (all models)** | 이번 주 전체 모델 한도 사용률 |
| **Current week (Sonnet only)** | 이번 주 Sonnet 모델 한도 사용률 |

**사용 패턴 분석**

지난 24시간 동안 내 사용량에서 어떤 패턴이 주요 원인인지 알려줘요.\
서브에이전트를 많이 썼는지, 특정 기능이 많은 요청을 차지했는지 등을 보여줘요.

<Callout type="info">
  사용률은 이 기기의 로컬 세션 기준이에요.\
  다른 기기나 claude.ai에서 사용한 내역은 포함되지 않아요.
</Callout>

## 화면 전환

| 키 | 동작 |
|----|------|
| **d** | 일간(day) 보기로 전환 |
| **w** | 주간(week) 보기로 전환 |

## `/context`와 뭐가 달라요?

| | `/usage` | `/context` |
|--|----------|------------|
| 보여주는 것 | 비용·한도·세션 통계 | 현재 대화의 컨텍스트 창 사용률 |
| 언제 쓰나 | 비용·한도 파악할 때 | 대화 용량 파악할 때 |

## 관련 명령어

| 명령어 | 설명 |
|--------|------|
| `/context` | 현재 대화 컨텍스트 사용률 시각적으로 확인 |
| `/compact` | 대화 요약해서 토큰 절약 |
