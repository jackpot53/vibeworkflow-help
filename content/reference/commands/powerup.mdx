## 언제 쓰나요?

Claude Code를 처음 써보거나, 아직 써보지 못한 기능이 궁금할 때 사용해요.
터미널을 벗어나지 않고, 지금 바로 배울 수 있어요.

**이런 상황에서 쓰세요:**

- Hooks, MCP, 서브에이전트 같은 고급 기능을 처음 배우고 싶을 때
- 문서를 찾아다니지 않고 터미널 안에서 바로 개념을 익히고 싶을 때
- 내가 쓰지 않던 기능을 빠르게 훑어보고 싶을 때

<Callout type="info">
  v2.1.90 이상에서 사용 가능하며, 모든 플랜에서 무료로 이용할 수 있어요.
</Callout>

---

## 사용법

```
> /powerup
```

실행하면 터미널 안에서 인터랙티브 튜토리얼이 시작됩니다.
애니메이션 데모를 보며 각 기능을 단계별로 따라갈 수 있어요.

---

## 배울 수 있는 내용

| 주제 | 설명 |
|------|------|
| **Hooks** | 특정 이벤트에 자동으로 실행할 동작 설정 |
| **MCP** | 외부 도구를 Claude에 연결하는 방법 |
| **서브에이전트** | 복잡한 작업을 병렬로 처리하는 방법 |
| **컨텍스트 관리** | 긴 세션에서 컨텍스트를 효율적으로 관리하는 방법 |
| **Plan Mode** | 코드 작성 전에 계획 먼저 세우는 방법 |
| **Rewind** | 대화와 코드를 이전 시점으로 되돌리는 방법 |
| **Skills** | 나만의 커스텀 명령어 만드는 방법 |

<Callout type="tip">
  Claude Code에는 수많은 기능이 있지만, 대부분의 사용자는 20% 정도만 활용한다고 해요.\
  `/powerup`으로 아직 써보지 못한 기능들을 발견해보세요.
</Callout>
