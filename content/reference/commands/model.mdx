## 언제 쓰나요?

기본 모델보다 더 강력한 모델이 필요할 때, 또는 응답 속도를 높이고 싶을 때 사용해요.

<Callout type="info">
  `/model`로 변경한 설정은 **현재 세션과 이후 세션 모두에 유지**돼요.\
  특정 모델 ID를 직접 지정하려면 `--model` 플래그를 사용해요.
</Callout>

---

## 사용법

```
> /model
```

인자 없이 실행하면 아래와 같은 선택 메뉴가 나타나요:

```
Select model
Switch between Claude models. Applies to this session and future Claude Code
sessions. For other/previous model names, specify with --model.

  1. Default (recommended)  Opus 4.7 with 1M context · Most capable for
                            complex work
  2. Sonnet                 Sonnet 4.6 · Best for everyday tasks
  3. Haiku                  Haiku 4.5 · Fastest for quick answers
❯ 4. Opus Plan Mode ✔       Use Opus in plan mode, Sonnet otherwise
```

방향키로 선택하고 Enter를 누르면 바로 적용돼요.

---

## 모델 옵션

| 번호 | 이름 | 모델 ID | 특징 |
|------|------|---------|------|
| 1 | **Default** | claude-opus-4-7 (1M context) | 복잡한 작업에 가장 강력함 |
| 2 | **Sonnet** | claude-sonnet-4-6 | 일상적인 작업에 최적 (기본값) |
| 3 | **Haiku** | claude-haiku-4-5 | 빠른 답변이 필요할 때 |
| 4 | **Opus Plan Mode** | Opus (계획) + Sonnet (실행) | 설계는 Opus, 구현은 Sonnet |

<Callout type="tip">
  **Opus Plan Mode**는 `/plan` 단계에선 Opus를 쓰고, 실제 코드 작성 단계에선 Sonnet을 써요.\
  품질과 속도를 동시에 잡고 싶을 때 추천해요.
</Callout>

---

## 모델 ID 직접 지정

이전 버전 모델이나 목록에 없는 모델은 `--model` 플래그로 지정해요:

```bash
claude --model claude-opus-4-5
claude --model claude-haiku-4-5-20251001
```
