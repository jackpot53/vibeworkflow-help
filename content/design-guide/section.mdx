# 페이지 단위: 섹션(Section)

**섹션**은 페이지를 세로로 나누는 큰 구역이에요.\
스크롤하면 순서대로 지나치는 각각의 덩어리를 섹션이라고 불러요.

<SectionMap />

| 섹션 이름 | 역할 |
|-----------|------|
| **Navbar** | 상단 고정 내비게이션 바 |
| **Hero** | 페이지 최상단. 핵심 메시지와 CTA 버튼이 있어요 |
| **Features** | 서비스의 주요 기능을 나열하는 섹션 |
| **How it works** | 사용 방법이나 흐름을 설명하는 섹션 |
| **Pricing** | 요금제를 비교하는 섹션 |
| **Testimonials** | 사용자 후기나 인용문을 보여주는 섹션 |
| **FAQ** | 자주 묻는 질문과 답변 섹션 |
| **CTA** | 행동 유도 섹션 — "지금 시작하기" 같은 버튼이 크게 있어요 |
| **Footer** | 페이지 맨 아래. 링크, 저작권 정보 등 |

<Callout type="tip">
"Hero 섹션 배경색 바꿔줘", "Pricing 섹션 카드 3개로 만들어줘"처럼 섹션 이름을 쓰면 Claude가 정확히 찾아가요.
</Callout>

<Callout type="info">
각 섹션별 더 다양한 디자인 예시는 **[prebuiltui.com/components](https://prebuiltui.com/components)** 에서 볼 수 있어요. 마음에 드는 디자인을 캡처하거나 코드를 복사해서 Claude에게 넘겨주세요.
</Callout>
