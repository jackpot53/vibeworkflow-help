# 페이지 단위: 섹션(Section)

**섹션**은 페이지를 세로로 나누는 큰 구역이에요.\
스크롤하면 순서대로 지나치는 각각의 덩어리를 섹션이라고 불러요.

<SectionMap />

| 섹션 이름 | 역할 |
|-----------|------|
| **Navbar** | 상단 고정 내비게이션 바 |
| **Hero** | 페이지 최상단. 핵심 메시지와 CTA 버튼이 있어요 |
| **Features** | 서비스의 주요 기능을 나열하는 섹션 |
| **How it works** | 사용 방법이나 흐름을 설명하는 섹션 |
| **Pricing** | 요금제를 비교하는 섹션 |
| **Testimonials** | 사용자 후기나 인용문을 보여주는 섹션 |
| **FAQ** | 자주 묻는 질문과 답변 섹션 |
| **CTA** | 행동 유도 섹션 — "지금 시작하기" 같은 버튼이 크게 있어요 |
| **Footer** | 페이지 맨 아래. 링크, 저작권 정보 등 |

<Callout type="tip">
"Hero 섹션 배경색 바꿔줘", "Pricing 섹션 카드 3개로 만들어줘"처럼 섹션 이름을 쓰면 Claude가 정확히 찾아가요.
</Callout>

<Callout type="info">
각 섹션별 실제 디자인 예시는 **[prebuiltui.com/components](https://prebuiltui.com/components)** 에서 볼 수 있어요. Tailwind CSS 기반 코드를 그대로 복사해서 Claude에게 "이 디자인과 비슷하게 만들어줘"라고 넘겨주세요.
</Callout>

## 섹션별 예시 코드

각 섹션의 기본 구조를 Tailwind CSS 코드로 보여드릴게요.\
Claude에게 이 코드를 그대로 주거나, [prebuiltui.com](https://prebuiltui.com/components)에서 마음에 드는 변형을 골라 전달하면 돼요.

### Navbar

상단에 고정되는 내비게이션 바예요. 로고, 메뉴 링크, CTA 버튼으로 구성돼요.

[prebuiltui에서 Navbar 예시 보기 →](https://prebuiltui.com/components/navbar)

```html
<nav class="w-full flex justify-between items-center px-8 py-4 bg-white border-b border-gray-100">
  <span class="text-xl font-bold">MyApp</span>
  <div class="hidden md:flex items-center gap-8 text-sm text-gray-600">
    <a href="#" class="hover:text-gray-900">기능</a>
    <a href="#" class="hover:text-gray-900">요금</a>
    <a href="#" class="hover:text-gray-900">문의</a>
  </div>
  <button class="bg-black text-white text-sm px-4 py-2 rounded-lg">시작하기</button>
</nav>
```
