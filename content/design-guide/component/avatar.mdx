# Avatar — 아바타

**Avatar**는 사용자 프로필 이미지나 이니셜을 원형으로 표시하는 컴포넌트예요.\
이미지가 없거나 로드에 실패하면 이니셜(텍스트 폴백)을 자동으로 보여줘요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 댓글·게시물 작성자 표시 | 게시판, 피드, 채팅 |
| 로그인한 유저를 헤더에 표시 | 네비게이션 바 오른쪽 |
| 팀 멤버 목록 | 프로젝트 대시보드 |
| 여러 명을 겹쳐 표시 | 공동 작업자 미리보기 (Avatar Group) |
