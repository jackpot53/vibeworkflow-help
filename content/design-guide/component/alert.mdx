# Alert — 알림

**Alert**는 사용자가 주의를 기울여야 할 정보를 페이지 안에 인라인으로 보여주는 컴포넌트예요.\
아이콘, 제목, 설명, 액션 버튼을 자유롭게 조합할 수 있어요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 일반 안내·정보 메시지 | 새 기능 출시, 설정 저장 완료 |
| 성공·완료 메시지 | 배포 완료, 결제 성공 |
| 주의·경고 메시지 | 되돌릴 수 없는 작업, 세션 만료 예정 |
| 오류·실패 메시지 | 요청 실패, 권한 없음 |
