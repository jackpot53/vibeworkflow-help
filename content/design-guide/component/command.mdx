# Command — 커맨드 팔레트

**Command**는 검색과 명령 실행을 하나로 합친 컴포넌트예요.\
텍스트를 입력하면 관련 항목을 즉시 필터링하고, 키보드로 선택해서 빠르게 실행할 수 있어요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 앱 전역 검색 | ⌘K로 열리는 커맨드 팔레트 |
| 항목 빠른 탐색 | 파일·페이지·팀원 검색 |
| 액션 실행 | 새 파일 만들기, 브랜치 전환 |
| 필터 UI | Combobox, 태그 선택기 |

## 구성 요소

| 컴포넌트 | 역할 |
|----------|------|
| `Command` | 루트 컨테이너 |
| `CommandInput` | 검색 입력창 (돋보기 아이콘 포함) |
| `CommandList` | 항목 목록 스크롤 영역 |
| `CommandEmpty` | 검색 결과 없을 때 메시지 |
| `CommandGroup` | 항목을 섹션으로 묶기. `heading`으로 제목 표시 |
| `CommandItem` | 개별 선택 가능한 항목 |
| `CommandShortcut` | 항목 오른쪽 단축키 힌트 |
| `CommandSeparator` | 그룹 사이 구분선 |
| `CommandDialog` | Dialog 안에 Command를 감싸는 래퍼 |

## 기본 커맨드

검색창에 입력하면 항목이 즉시 필터링돼요. `CommandGroup`으로 파일·설정 섹션을 나누고, `CommandShortcut`으로 단축키 힌트를 보여줘요.

<CommandBasicDemo />

<VibeBlock>
```
shadcn Command로 파일·설정 명령어를 검색할 수 있는 기본 팔레트를 만들어줘.
각 항목에 lucide 아이콘과 키보드 단축키 힌트를 넣어줘.
```
</VibeBlock>

## 커맨드 다이얼로그 (⌘K)

`CommandDialog`로 감싸면 모달로 띄울 수 있어요.\
`useEffect`로 `⌘K` (Mac) / `Ctrl+K` (Windows) 단축키를 연결하는 게 일반적이에요.

<CommandDialogDemo />

<VibeBlock>
```
shadcn CommandDialog로 ⌘K 단축키로 열리는 커맨드 팔레트를 만들어줘.
"빠른 실행"과 "도구" 그룹으로 나누고, 버튼을 눌러도 열리게 해줘.
```
</VibeBlock>

<Callout type="tip">
`useEffect`로 `keydown` 이벤트를 등록해 `⌘K`를 연결해요. `e.preventDefault()`를 빠뜨리면 브라우저 기본 동작(주소창 포커스 등)과 충돌해요.
</Callout>

## 그룹 커맨드

`CommandGroup`을 여러 개 쌓아 항목을 카테고리별로 분류해요.\
프로젝트·팀원·데이터베이스처럼 서로 다른 종류의 항목을 한 화면에서 검색할 때 유용해요.

<CommandGroupDemo />

<VibeBlock>
```
shadcn Command로 프로젝트·팀원·DB를 한 번에 검색하는 팔레트를 만들어줘.
각 카테고리를 CommandGroup으로 나누고, 카테고리마다 다른 lucide 아이콘을 써줘.
```
</VibeBlock>
