# Card — 카드

**Card**는 관련 정보를 하나의 영역으로 묶어 표시하는 컨테이너예요.\
Header, Content, Footer 구조로 구성되며, 다양한 콘텐츠를 자유롭게 담을 수 있어요.

<CardWhenDemo />

## 구성 요소

| 컴포넌트 | 역할 |
|----------|------|
| `Card` | 전체 컨테이너. `size="sm"`으로 더 작은 버전 사용 가능 |
| `CardHeader` | 제목·설명·액션을 담는 상단 영역 |
| `CardTitle` | 카드 제목 |
| `CardDescription` | 부제목 또는 설명 텍스트 |
| `CardAction` | 헤더 우측에 배치되는 액션 영역 (버튼, 메뉴 등) |
| `CardContent` | 본문 콘텐츠 영역 |
| `CardFooter` | 하단 버튼·링크 영역. 배경이 살짝 다른 색으로 구분돼요 |

제목은 "알림 설정", 설명은 "이메일과 푸시 알림 수신 여부를 관리하세요."
본문엔 체크 아이콘 목록으로 "새 댓글 알림"(초록), "멘션 알림"(초록),

푸터엔 "저장" 버튼(기본)이랑 "취소" 버튼(outline) 나란히 달아줘.
```
</VibeBlock>

### 헤더 액션

<CardActionDemo />

<VibeBlock>
```
shadcn Card로 팀 멤버 카드 만들어줘.
제목 "팀 멤버", 설명 "현재 프로젝트에 참여 중인 멤버예요.",
