# Badge — 뱃지

**Badge**는 상태·카테고리·카운트 같은 짧은 정보를 강조해서 표시하는 컴포넌트예요.\
텍스트나 아이콘과 함께 조합하면 한눈에 의미를 전달할 수 있어요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 새 기능·버전 표시 | NEW, BETA, v2.0 |
| 콘텐츠 상태 | LIVE, DEPRECATED, 진행 중 |
| 카테고리·태그 | 프론트엔드, 디자인, 공지 |
| 알림 카운트 | 읽지 않은 메시지 수 |

## Variant

`variant` prop으로 뱃지의 시각적 스타일을 고를 수 있어요. 생략하면 `default`가 적용돼요.
