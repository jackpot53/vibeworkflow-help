# Carousel — 캐러셀

**Carousel**은 여러 콘텐츠를 슬라이드 형태로 넘겨볼 수 있는 컴포넌트예요.\
화살표 버튼이나 키보드 방향키로 이전·다음 슬라이드를 탐색할 수 있어요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 이미지 갤러리 | 상품 사진 여러 장 |
| 카드 목록 | 기능 소개, 팀 멤버 |
| 후기·인용 | 사용자 리뷰, 추천사 |
| 공지·배너 | 메인 페이지 히어로 배너 |

## 기본 캐러셀

숫자 카드 5장을 넘겨볼 수 있어요. 화살표 버튼이나 키보드 ← → 로 이동해요.

<CarouselBasicDemo />

<VibeBlock>
```
shadcn Carousel로 숫자 1~5를 보여주는 기본 슬라이드를 만들어줘.
이전·다음 버튼을 포함해줘.
```
</VibeBlock>

## 한 화면에 여러 슬라이드

`basis-1/2`로 한 번에 2개씩 보이는 슬라이드예요.\
`opts={{ align: 'start' }}`로 슬라이드가 왼쪽 기준으로 정렬돼요.

<CarouselSizeDemo />

<VibeBlock>
```
shadcn Carousel에서 한 화면에 카드 2개씩 보이게 만들어줘.
각 카드에 태그 뱃지와 제목을 넣어줘.
```
</VibeBlock>

## 후기 카드 슬라이드

사용자 후기를 한 장씩 넘겨볼 수 있어요. 별점, 후기 텍스트, 이름, 역할로 구성해요.

<CarouselCardDemo />

<VibeBlock>
```
shadcn Carousel로 사용자 후기 슬라이드를 만들어줘.
카드마다 별점, 후기 내용, 이름, 직책을 보여줘.
```
</VibeBlock>
