# Tabs — 탭

**Tabs**는 하나의 공간에서 여러 콘텐츠 섹션을 탭 버튼으로 전환하는 컴포넌트예요.\
`TabsList` → `TabsTrigger` → `TabsContent` 구조로 구성되며, 페이지 이동 없이 관련 콘텐츠를 깔끔하게 분리해줘요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 연관된 콘텐츠 그룹화 | 개요·활동·설정처럼 같은 맥락의 뷰 분리 |
| 설정 카테고리 분류 | 계정·알림·보안 설정 탭 |
| 데이터 뷰 전환 | 통계 차트 ↔ 목록 테이블 |
| 코드·미리보기 | 에디터와 렌더링 결과 나란히 |

## 구성 요소

| 컴포넌트 | 역할 |
|----------|------|
| `Tabs` | 루트 컨테이너. `defaultValue`로 초기 탭 지정 |
| `TabsList` | 탭 버튼들을 담는 가로 바 |
| `TabsTrigger` | 개별 탭 버튼. `value`로 연결된 패널 식별 |
| `TabsContent` | 각 탭에 대응하는 콘텐츠 패널 |
