# Sidebar — 사이드바

**Sidebar**는 앱의 주요 메뉴와 네비게이션을 담는 사이드 패널이에요.\
아이콘·레이블·뱃지를 조합한 메뉴 항목, 섹션 그룹, 접힘 토글을 지원해요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 앱의 주요 네비게이션 | 대시보드, 프로젝트, 설정 |
| 섹션별 메뉴 구분 | 메인 메뉴 / 관리 / 개인 |
| 화면 공간 절약 | 아이콘 전용 모드로 접기 |
| 다단계 메뉴 | 서브메뉴 펼치기/접기 |

### 그룹 섹션

<SidebarGroupDemo />

<VibeBlock>
```
사이드바를 두 섹션으로 나눠줘.
"메인 메뉴" 그룹엔 개요, 분석(활성), 보고서, 메시지(뱃지 5),

사이드바 하단엔 사용자 이름 "김지수" 표시해줘.
구분선으로 두 그룹 나눠줘.
```
</VibeBlock>
