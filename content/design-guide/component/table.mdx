# Table — 테이블

**Table**은 정형 데이터를 행과 열로 구조화해 표시하는 컴포넌트예요.\
헤더·본문·푸터 구조로 구성되며, 목록 비교·현황 파악·청구 내역처럼 데이터를 스캔해야 할 때 적합해요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 정형 데이터 목록 | 팀원·고객·상품 목록 |
| 상태 현황 파악 | API 상태·배포 현황·서비스 모니터링 |
| 항목 비교 | 요금제·스펙 비교표 |
| 청구·정산 내역 | 주문 내역·인보이스·영수증 |

## 구성 요소

| 컴포넌트 | 역할 |
|----------|------|
| `Table` | 루트 컨테이너. `width: 100%`로 가득 채워짐 |
| `TableHeader` | `<thead>` 영역. `TableRow` + `TableHead` 조합 |
| `TableBody` | `<tbody>` 영역. 데이터 행들을 담음 |
| `TableFooter` | `<tfoot>` 영역. 합계·요약 행에 사용 |
| `TableRow` | 개별 행 (`<tr>`). 호버 시 하이라이트 |
| `TableHead` | 헤더 셀 (`<th>`). `text-right`로 숫자 열 정렬 가능 |
| `TableCell` | 데이터 셀 (`<td>`). `colSpan`으로 열 병합 가능 |
| `TableCaption` | 테이블 하단 캡션 텍스트 |

<Callout type="tip">
숫자·금액 열은 `TableHead`와 `TableCell` 모두에 `className="text-right"`를 주면 헤더와 데이터 정렬이 맞아떨어져요.
</Callout>
