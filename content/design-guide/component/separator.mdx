# Separator — 구분선

**Separator**는 콘텐츠 사이에 얇은 선을 그어 영역을 나누는 컴포넌트예요.\
가로선(horizontal)과 세로선(vertical) 두 방향을 모두 지원해요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 섹션 사이 시각적 분리 | 프로필 정보 / 통계 사이 |
| 드롭다운·메뉴 그룹 구분 | 계정 설정 / 로그아웃 사이 |
| 인라인 항목 분리 | 블로그 · 문서 · 소스코드 |
| 카드 내부 영역 구분 | 헤더 / 본문 / 푸터 사이 |

## 방향 (orientation)

`orientation` prop으로 가로선(`horizontal`)과 세로선(`vertical`)을 선택할 수 있어요.\
생략하면 `horizontal`이 기본값이에요.

<SeparatorOrientationDemo />

<VibeBlock>
```
shadcn Separator로 가로선과 세로선 예시를 각각 보여줘.
세로선은 "블로그", "문서", "소스코드" 텍스트 사이에 넣어줘.
```
</VibeBlock>
