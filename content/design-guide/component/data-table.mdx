# Data Table — 데이터 테이블

**Data Table**은 TanStack Table을 기반으로 정렬·필터링·페이지네이션·열 표시 토글을 지원하는 인터랙티브 테이블이에요.\
기본 `Table` 컴포넌트 위에 상태 관리 레이어를 얹는 구조라 원하는 기능만 골라서 붙일 수 있어요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 대량 데이터 탐색 | 작업 목록, 사용자 관리, 주문 내역 |
| 필터·검색이 필요한 목록 | 이름·상태로 좁혀보기 |
| 열 정렬이 필요한 표 | 날짜순, 우선순위순 정렬 |
| 열을 선택적으로 숨기고 싶은 표 | 컴팩트 뷰 전환 |

## 구성 요소

| 항목 | 역할 | 예시 |
|------|------|------|
| `columns` 정의 | `ColumnDef<T>[]` — 각 열의 렌더링·정렬·필터 방식 선언 | <DataTableColumnsDemo /> |
| `useReactTable` | TanStack Table 인스턴스 생성. 필요한 Row Model을 조합해 기능을 활성화 | |
| 툴바 | 검색 `Input` + 열 표시·숨김 `DropdownMenu` | <DataTableToolbarDemo /> |
| `flexRender` | `header`·`cell` 정의를 실제 JSX로 렌더링 | |
| 페이지네이션 | `previousPage()` / `nextPage()` + 현재 위치 표시 | <DataTablePaginationDemo /> |
