# Collapsible — 접기/펼치기

**Collapsible**은 콘텐츠를 접고 펼칠 수 있는 토글 컴포넌트예요.\
기본적으로 숨겨둘 내용을 필요할 때만 보이게 해 화면을 깔끔하게 유지해줘요.

## 언제 써요?

| 상황 | 예시 |
|------|------|
| 자주 묻는 질문 | FAQ 섹션 |
| 추가 옵션·필터 숨기기 | 검색 필터, 고급 설정 |
| 설정 항목 그룹화 | 알림, 개인정보, 계정 설정 패널 |
| 긴 콘텐츠 요약 | 더보기/접기 버튼 |
